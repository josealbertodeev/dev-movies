## 📋 Sobre o Projeto

Dev Movies é uma aplicação web que permite aos usuários explorar informações sobre filmes. O projeto utiliza tecnologias modernas como React, Vite para um desenvolvimento rápido, e Styled Components para estilização.

## ✨ Funcionalidades

- 🎥 Navegação de filmes
- 🔍 Busca de filmes
- 📱 Design responsivo
- ⚡ Performance otimizada
- 🎨 Interface moderna e intuitiva

## 🚀 Tecnologias Utilizadas

- **React** - Biblioteca JavaScript para construção de interfaces
- **Vite** - Build tool moderna e rápida
- **React Router DOM** - Navegação entre páginas
- **Styled Components** - Estilização com CSS-in-JS
- **Axios** - Cliente HTTP para requisições
- **Swiper** - Carrossel/slider de componentes
- **ESLint** - Linter para qualidade de código
- **Prettier** - Formatação de código

## 📦 Pré-requisitos

Antes de começar, você precisa ter instalado em sua máquina:

- [Node.js](https://nodejs.org/) (versão 16 ou superior)
- [npm](https://www.npmjs.com/) ou [yarn](https://yarnpkg.com/)

## 🔧 Instalação

1. Clone o repositório:
```bash
git clone https://github.com/seu-usuario/dev-movies.git
```

2. Entre na pasta do projeto:
```bash
cd dev-movies
```

3. Instale as dependências:
```bash
npm install
```
ou
```bash
yarn install
```

## 🎮 Como Usar

### Modo Desenvolvimento

Para iniciar o servidor de desenvolvimento:

```bash
npm run dev
```
ou
```bash
yarn dev
```

A aplicação estará disponível em `http://localhost:5173`

### Build para Produção

Para criar uma versão otimizada para produção:

```bash
npm run build
```
ou
```bash
yarn build
```

### Preview da Build

Para visualizar a versão de produção localmente:

```bash
npm run preview
```
ou
```bash
yarn preview
```

### Lint

Para verificar a qualidade do código:

```bash
npm run lint
```
ou
```bash
yarn lint
```

## 📁 Estrutura do Projeto

```
dev-movies/
├── src/
│   ├── assets/        # Imagens, ícones e recursos estáticos
│   ├── components/    # Componentes reutilizáveis

```

## 🎨 Padrões de Código

O projeto utiliza ESLint e Prettier para manter a qualidade e consistência do código:

- **ESLint**: Verifica erros e padrões de código
- **Prettier**: Formata o código automaticamente
- **Import Helpers**: Organiza os imports automaticamente
